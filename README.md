## Описание программы

Random Password Generator (Генератор случайных паролей) — это программное приложение, предназначенное для создания надёжных паролей на основе заданных пользователем параметров.

## Возможности:

- Гибкая настройка длины
- Детальный контроль состава символов
- Копирование в буфер обмена
- Сохранение истории
- Генерация нескольких паролей
- Оценка сложности пароля

## Инструкция по запуску

### Запуск:
1. Откройте терминал или командную строку
2. Перейдите в папку, где лежит ваш файл
3. Введите команду для запуска и нажмите Enter:

## Примеры использования:

### Создание пароля для нового аккаунта на сайте
1. Длина: Установите значение 12
2. Состав: Убедитесь, что все флажки включены: «Строчные буквы», «Заглавные буквы», «Цифры», «Специальные символы».
3. Генерация: Нажмите кнопку «Сгенерировать».
4. Генератор может выдать пароль вида
