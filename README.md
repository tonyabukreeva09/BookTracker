# Антонина Букреева
Это простое графическое приложение (GUI) для генерации случайных паролей. Программа позволяет пользователю гибко настраивать параметры будущего пароля: выбирать его длину и состав символов (строчные и прописные буквы, цифры, специальные символы).

Все сгенерированные пароли автоматически сохраняются в историю, которая сохраняется в файл history.json и отображается в виде таблицы в интерфейсе приложения. Также доступна функция копирования пароля в буфер обмена.

Проект написан на языке Python с использованием стандартной библиотеки tkinter для создания интерфейса и json для работы с файлами.
Тест на минимальную длину:
Установите ползунок длины на значение 4.
Оставьте включенным только флажок «Цифры».
Нажмите «Сгенерировать».
Ожидаемый результат: В поле ввода появится строка из 4 цифр (например, 1829).
Тест на состав символов:
Установите длину на 10.
Включите только флажки «Большие буквы» и «Маленькие буквы».
Нажмите «Сгенерировать».
Ожидаемый результат: Пароль будет состоять из 10 латинских букв разного регистра (например, kLpZmQwRtS).
Тест на историю:
Сгенерируйте три разных пароля с разными настройками.
Посмотрите на таблицу «История» внизу окна.
Ожидаемый результат: В таблице будут отображены все три сгенерированных пароля в порядке их создания.
